# Two-Phase-Bipolar-Stepper-Motor-Controller-KiCad-
 2-phase bipolar stepper motor driver PCB designed in KiCad. Uses L297 stepper sequencer + L298N dual H-bridge driver with 8x Schottky flyback diodes. Supports STEP/DIR interface, 2A/channel drive current, screw terminal I/O. Gerber files included.
